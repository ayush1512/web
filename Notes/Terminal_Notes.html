Terminal Commands:

ls command

The `ls` command is a command-line utility for listing the contents of a directory.

pwd command

The pwd command is a command-line utility that stands for "print working directory".

cd command

The cd command is a command-line utility that stands for "change directory".

  • cd ~ - Moves one level up in directory.
  • cd .. - Moves to the home directory.
  • cd / - This command will change the current directory to the root directory.
  • cd . - This command will keep you in the current directory (. represents the current directory).
  • cd - - This command will change the directory to the previous directory you were in.
  • cd $HOME - This command will change the current directory to the home directory of the current user. $HOME is an environment variable that represents the home directory.
  • cd directory/subdirectory - This command will change the current directory to a subdirectory. Replace directory/subdirectory with the actual path.
  • cd ~username - This command will change the current directory to the home directory of the specified user. Replace username with the actual username.

mkdir command

The mkdir command is a command-line utility that stands for "make directory".

mkdir -p /path/to/new_directory

This will create the directory new_directory and its parent directories (path and to) if they do not already exist.

man command

The man command is a command-line utility that stands for "manual".

touch command

The touch command is typically used to create new, empty files. It can also be used to change the timestamps on existing files and directories.

rm command

The rm command stands for "remove". It is used to delete files and directories from the file system.

rmdir command

The rmdir command 'works on empty directories'.
